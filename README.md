# Docker stack for MQTT purpose
- Uses mosquitto as MQTT broker (see http://mosquitto.org/)

## Usage

* Start the stack
  * From local git repository

```console
developer@host:~/.../stack-mqtt$ docker-compose up -d
```

* Stop the stack
  * From local git repository

```console
developer@host:~/.../stack-mqtt$ docker-compose down
```

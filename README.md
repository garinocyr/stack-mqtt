# Docker stack for MQTT purpose
- Uses mosquitto as MQTT broker (see http://mosquitto.org/)

## Usage

* Start the stack
  * From local git repository

```sh
developer@host:~/.../stack-mqtt$ podman-compose up -d
```

* Stop the stack
  * From local git repository

```sh
developer@host:~/.../stack-mqtt$ podman-compose down
```
